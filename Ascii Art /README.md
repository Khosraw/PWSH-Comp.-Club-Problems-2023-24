# ASCII Art 

Print out the following ASCII art of the queen chess piece!
 ```
     _(_)_      
    _/___\_  
     /___\   
    [|||||]       
     \___/       
     |||||     
     |___|
     |___|      
    /_____\     
   (_______)   
  (_________)
  /_________\
``` 

## Input Format:
N/A
## Output Format:
Print the ASCII art of the queen chess piece. Trailing spaces will be ignored.
